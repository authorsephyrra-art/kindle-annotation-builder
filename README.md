# Kindle Annotation Builder

Paste a passage spec block. Click Build. Export as PNG.

## Setup

1. Open `src/App.jsx`
2. Delete the placeholder code
3. Paste your full Kindle Annotation Builder code
4. Run `npm install`
5. Run `npm run dev`
6. Open the link in your browser

## Deploy to GitHub Pages

1. Create a new repo on GitHub called `kindle-annotation-builder`
2. Push this folder to that repo
3. Go to Settings > Pages > set Source to "GitHub Actions"
4. It deploys automatically on push

Your app will be at: `https://YOUR-USERNAME.github.io/kindle-annotation-builder/`
